# Saroj-AWS-Cloud-Practitioner-Preparation-Guide

This repository contains my structured study notes and practise exam while preparing for the **AWS Certified Cloud Practitioner (CLF-C02)** certification.

I created this guide to organize important AWS concepts in one place and to make revision easier before the exam. It also serves as a reference for anyone starting their cloud learning journey.

---

## Topics Covered

This repository includes notes on:

- Cloud Computing Fundamentals
- AWS Global Infrastructure
- Core AWS Services (EC2, S3, RDS, Lambda, VPC, etc.)
- Identity and Access Management (IAM)
- Shared Responsibility Model
- Security and Compliance
- AWS Pricing and Billing
- Well-Architected Framework
- Support Plans
- Exam-focused concepts and revision material

---

## Purpose of This Repository

I built this repository:

- while preparing for the AWS Cloud Practitioner certification
- to strengthen my cloud fundamentals
- to maintain structured revision notes
- to help other beginners preparing for CLF-C02
